## 下载

官网：<https://www.python.org/>

## 安装

- Windows：
直接使用Windows版本安装

- Linux：
  建议使用源码
  
    ```bash
        # ./configure
        # make && make install
    ```

## 设置环境变量

- Windows：
    - DOS命令：
        ```dos
        > path=%path%;C:\Python
        ```

    - 打开 **控制面板** 添加环境变量
- Linux：
    ```bash
        # export PATH="$PATH:/usr/local/bin/python"
    ```
