格式：

  • 空元组()
  • 含有元素的元组：
    • 带括号：(a,)，(a,b,c,d,...)

  • 定义含有一个元素的带括号的元组，应当在元素后面加,,以区分变量 = (值或表达式)把()作为运算符的定义方式

索引

格式：变量[下标]
索引方式：

  • 从左往右以0开始
  • 从右往左以-1开始

取值：不能越界

截取(切片)

格式：变量[start:stop[:step]]
取值：

  • 取值区间左闭右开，会自动处理越界索引
  • step默认值为1，为正顺序，为负逆序

迭代

格式：for x in (1,2,3):print(x,end=" ")
